# Design System — Artisan Handmade Jewelry

## Inspiration
Minimalist fashion e-commerce layout (VESSA-style): full-width hero with bold text overlay, clean product grid, structured sections. Adapted for handmade Ukrainian jewelry brand.

## Color Palette
- `--peach: #F4C2C2` — primary accent, buttons
- `--dusty-rose: #C08081` — hover states
- `--blush: #FFF5F5` — page background
- `--charcoal: #2D3142` — primary text, dark elements
- `--warm-gray: #8B7E74` — secondary text, meta
- `--silver: #C0C0C0` — borders, metallic accents
- `--midnight: #1A1A2E` — dark section backgrounds
- `--mint: #88D8B0` — success states

## Typography
- **Display**: "Playfair Display" (Google Fonts) — headings, hero
- **Body**: "Montserrat" (Google Fonts) — body text, UI
- H1: Playfair Display 700, 56px, lh 1.1
- H2: Playfair Display 600, 36px, lh 1.2
- H3: Montserrat 600, 20px
- Body: Montserrat 400, 16px, lh 1.6
- Small: Montserrat 400, 14px

## Layout
- Max container: 1280px, centered
- Grid: 12-column
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card border-radius: 16px
- Button border-radius: 24px (pill)

## Visual Style
- Clean white/blush backgrounds with peach accents
- Large editorial product photography
- Generous white space
- Thin hairline borders (#E5E5E5)
- Sticky nav: transparent on hero → white + shadow on scroll
- Hover: subtle lift (translateY -4px) + shadow
- Smooth transitions: 300ms ease

## Anti-patterns to avoid
- No heavy gradients
- No rounded-everything cards (keep some squared edges for elegance)
- No bold color fills on large sections (use sparingly)
